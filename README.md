FOAM-FSI
========

Installation
-----------

First, compile foam-extend-3.1, the nextRelease branch.

To compile the FSI library:

1. cd src/thirdParty
2. ./Allmake
3. cd -
4. ./Allwmake

Prerequisites
-----------

gcc 4.8 or higher due to C++11 features.
